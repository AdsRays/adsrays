# Instructions for ADSRays

## Environment
- Node 18.x

## Installation
- `npm ci`

## Development
- `npm run dev`

## Build
- `npm run build`

## Test
- `npm test` *(if tests are absent, this runs a stub)*

## Key Files
- `src/experiments/IntroPlayground.jsx`

## Style Guide
- Color palette: `#E2E6F5`, `#9D9E9E`, `#434242`, `#1877F2`
- Border radius: `5px`
